# Getting Started with Plexichat

Welcome to Plexichat! This guide will help you get started with the platform, whether you're a new user or exploring features for the first time.

## Creating Your Account

To begin using Plexichat, you'll need to create an account:

1. Navigate to your Plexichat instance (typically `https://your-domain.com`)
2. Click on "Register" or "Sign Up"
3. Provide the required information:
   - Username (unique identifier)
   - Email address (for account verification and notifications)
   - Password (strong, unique password recommended)
4. Complete any additional verification steps (CAPTCHA, email verification)
5. Accept the terms of service and privacy policy

## First Login

After creating your account:

1. Return to the login page
2. Enter your username and password
3. If two-factor authentication is enabled for your account:
   - Open your authenticator app (Google Authenticator, Authy, etc.)
   - Enter the 6-digit code displayed
4. Click "Login"

## Initial Setup Wizard

Upon first login, you may be guided through a setup wizard to:
- Set up your profile (display name, avatar, bio)
- Configure notification preferences
- Adjust privacy settings
- Join recommended servers or communities
- Set up two-factor authentication (if not already done)

## Navigation Overview

Plexichat's interface is organized into several key areas:

### Left Sidebar
- **Servers**: List of servers you're a member of
- **Direct Messages**: Your private conversations
- **Friends**: Online/offline status of your contacts
- **Profile**: Access to your user settings and profile

### Top Navigation
- **Search**: Global search across messages, users, and servers
- **Notifications**: Bell icon showing alerts and mentions
- **Settings**: Gear icon for user and server settings
- **User Menu**: Your avatar/icon for profile, status, and logout

### Main Content Area
- **Server View**: Channels and members when a server is selected
- **Chat View**: Message history and input when a channel is selected
- **Direct Message View**: Conversation history with selected user

## Basic Operations

### Sending Messages
1. Select a server from the left sidebar
2. Choose a text channel within that server
3. Type your message in the input box at the bottom
4. Press Enter or click the send button
5. Your message will appear in the chat history

### Adding Reactions
1. Hover over any message you want to react to
2. Click the "Add Reaction" emoji icon that appears
3. Select an emoji from the picker
4. The reaction will be added to the message

### Mentions
- Type `@` followed by a username to mention a specific user
- Type `@here` to mention all online members in the current channel
- Type `@everyone` to mention all members (if permissions allow)
- Type `@role` to mention all members with a specific role

### File Attachments
1. Click the paperclip icon in the message input
2. Select files from your device
3. Add an optional description
4. Press Enter to send the message with attachments

## Profile Customization

To customize your profile:
1. Click on your avatar in the bottom left
2. Select "Profile" from the menu
3. Click "Edit Profile"
4. Modify:
   - Display Name
   - Bio/About Me
   - Avatar/Image
   - Banner/Header Image
5. Save your changes

## Notification Settings

To adjust how you receive notifications:
1. Click on your avatar in the bottom left
2. Select "Settings"
3. Navigate to the "Notifications" tab
4. Configure:
   - Desktop notifications
   - Email notifications
   - Sound alerts
   - Notification content (mentions, direct messages, etc.)
   - Per-server notification overrides

## Privacy Controls

To manage your privacy:
1. Click on your avatar in the bottom left
2. Select "Settings"
3. Navigate to the "Privacy" tab
4. Adjust:
   - Who can send you direct messages
   - Who can see your online status
   - Who can see your activity
   - Data sharing preferences
   - Blocked users list
