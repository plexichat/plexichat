# Security Logout

You have been logged out of all sessions as a security precaution.

This usually happens for one of the following reasons:

1.  **Account Suspension:** An administrator has suspended your account.
2.  **Security Reset:** We detected suspicious activity and reset all sessions.
3.  **Manual Revocation:** You or an admin manually revoked your sessions.

## What to do next?

If you believe this was a mistake, please contact support or an administrator.

If you are an administrator and need to restore access, please log in with your credentials.
