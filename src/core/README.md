# Core Modules

Business logic modules for Plexichat.

## Modules

- `applications/` - Bot/application management, commands, OAuth
- `auth/` - Authentication, sessions, permissions, 2FA
- `automod/` - Automated moderation with rules and AI
- `database/` - Database connections and caching
- `embeds/` - Rich embed creation and validation
- `events/` - Event dispatching and routing
- `media/` - File uploads, processing, and storage
- `messaging/` - Direct messages and conversations
- `notifications/` - Notification preferences and delivery
- `polls/` - Poll creation and voting
- `presence/` - User online status and activities
- `ratelimit/` - Rate limiting middleware and storage
- `reactions/` - Message reactions (emoji)
- `relationships/` - Friends and blocking
- `search/` - Full-text search with multiple backends
- `servers/` - Server/guild management
- `settings/` - User settings storage
- `soundboard/` - Soundboard sounds
- `stickers/` - Sticker packs and usage
- `threads/` - Thread channels
- `voice/` - Voice/video chat with WebRTC
- `webhooks/` - Webhook management and execution

Each module has its own README with detailed documentation.
