# Applications Tests

Tests for bot/application functionality.

## Test Files

- `test_application_crud.py` - Application create/read/update/delete
- `test_command_options.py` - Command option validation
- `test_command_registration.py` - Command registration
- `test_components.py` - UI component tests
- `test_installation.py` - App installation flow
- `test_integration.py` - Integration tests
- `test_interaction_handling.py` - Interaction processing
- `test_oauth_flows.py` - OAuth2 flow tests
- `test_rate_limits.py` - App rate limiting
- `test_responses.py` - Interaction responses
- `test_webhooks.py` - App webhook tests
