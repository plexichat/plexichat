# Tests

Unit and integration tests for the common_utils library.

## Structure

- `config/` - Tests for the ConfigLoader utility
- `integration/` - Integration tests for cross-module functionality
- `logger/` - Tests for the Logger utility
- `validator/` - Tests for the Validator utility
- `version/` - Tests for the Version utility

## Running Tests

```bash
pytest tests/
```

## Test Coverage

Each subdirectory contains tests for its corresponding utility module in `utils/`.
