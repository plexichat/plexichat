# Version Tests

Unit tests for the Version utility.

## Test Coverage

- `test_version.py` - Tests for version parsing, formatting, comparison, compatibility, and increment operations

## Key Test Cases

- Parsing version strings (alpha, beta, candidate, release stages)
- Invalid version string handling
- Version formatting and roundtrip
- Version comparison (stage, major, minor, build)
- Compatibility checking
- Release line detection
- Version increment operations (build, minor, major, stage)
